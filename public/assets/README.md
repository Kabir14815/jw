# The House of Garg – Assets

Place brand assets here. They are served at `/assets/`.

| File | Use |
|------|-----|
| `logo.png` | Header logo (THG emblem + "THE HOUSE OF GARG"). If missing, text fallback is shown. |
| `intro-doors.png` | Animated intro (architectural doors + THG monogram). Used by IntroScreen. |
| `hero-banner.jpg` | Optional hero background (e.g. 1920×800px). |

**Images from “The house of garg Presentation.pdf”**  
Export images from the PDF (e.g. open in viewer → right‑click/save image, or use a PDF-to-image tool) and save them here with clear names so you can use them in hero, store section, or about block, e.g.:
- `presentation-store.jpg` – store / boutique visuals  
- `presentation-jewellery.jpg` – jewellery hero or section  
- `presentation-fashion.jpg` – fashion / couture section  

**Exodus font:** Save webfont as `public/Exodus.woff2`, then in `src/index.css` uncomment the `@font-face` and set `--font-heading` to `'Exodus', serif`.
